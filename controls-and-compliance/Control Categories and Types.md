# Control Categories

Cybersecurity controls are grouped into three main categories:

## 1. Administrative / Managerial Controls
- Focus on the human component of cybersecurity  
- Include policies, procedures, and defined employee responsibilities  
- Examples: data handling policies, role-based access assignments, security training  

## 2. Technical Controls
- Implemented through technology solutions  
- Examples: firewalls, intrusion detection systems (IDS), intrusion prevention systems (IPS), antivirus software, encryption  

## 3. Physical / Operational Controls
- Protect physical assets from unauthorized access  
- Examples: door locks, cabinet locks, surveillance cameras, badge readers


# Control Types

Controls can be further classified by type:

1. **Preventative Controls** – Prevent an incident from occurring  
2. **Corrective Controls** – Restore an asset after an incident  
3. **Detective Controls** – Determine if an incident has occurred or is in progress  
4. **Deterrent Controls** – Discourage attacks
